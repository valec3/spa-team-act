# Capa geoespacial — Límites distritales Perú

## Fuente
- **Shapefile**: INEI 2023 (Distrital)
- **Obtenido de**: geogpsperu.com (SuyoPomalia)
- **Fecha de descarga**: Junio 2026
- **Formato**: ESRI Shapefile (.shp + .dbf + .prj + .shx)
- **Cobertura**: 1.891 distritos del Perú
- **Columna clave**: `UBIGEO` (6 dígitos)

## Uso en el pipeline
Cargado por `src/01_carga.R` para joinear los datos distritales del Mapa de Pobreza con la capa vectorial.

## NOTA
Este shapefile es necesario para ejecutar el pipeline. Si se pierde, descargar nuevamente de geogpsperu.com o del IGN.
